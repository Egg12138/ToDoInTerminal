# ToDoInTerminal
A simple TODO application for users.



## 功能预期

用户态初始化（配置数据从存储位置， 提供默认值）

* INIT
* CONFIGs

日常使用

* ADD
* REMOVE
* SHOW
* CHANGE
* SET_SCORES
* STATISTICS
* CALENDER VIEW (图形界面的事情，参考ios的那个日历视图)

本地账号登出/登入

* LOGOUT
* LOGIN

同步

* 提供手动Sync借口
* 

文件加密，密码访问

* PWD访问
* SSH访问

* 桌面显示（Windows &一些主流Linux发行版的主流桌面）
